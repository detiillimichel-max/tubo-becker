# tubo-becker – Desafio das Cores

Jogo de puzzle estilo Water Sort feito em HTML5, 100% offline.

## Como jogar
1. Toque em um tubo para selecionar
2. Toque em outro tubo para despejar a cor
3. Só pode despejar se a cor de cima for igual e houver espaço
4. Complete separando todas as cores

## Recursos
- 10 níveis artesanais (do Fácil ao Médio)
- Nível 10 = BÔNUS: +99 vidas + 50 dicas + tubo extra liberado
- Após o nível 10: fases infinitas geradas aleatoriamente
- 10 dicas por fase
- Botão Desfazer ilimitado
- Salva progresso automaticamente

## Instalação
1. Baixe o arquivo `index.html`
2. Renomeie para `index.html` se necessário
3. Envie para seu GitHub Pages, Netlify ou abra direto no navegador

Feito para celular – não precisa instalar nada.
